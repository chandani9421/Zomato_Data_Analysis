#  Zomato Data Analysis Project

## Project Overview
This project focuses on analyzing Zomato restaurant data to extract meaningful insights about customer preferences, restaurant trends, and pricing patterns.

The goal is to perform Exploratory Data Analysis (EDA) and derive business insights that can help in decision-making.

## Objectives
* Understand restaurant distribution across locations
* Analyze pricing trends
* Identify popular cuisines
* Explore customer ratings and reviews
* Find factors affecting restaurant success

## Dataset Description
The dataset contains information such as:
* Restaurant Name
* Location
* Cuisines
* Average Cost for Two
* Ratings
* Votes
* Online Delivery Availability

## Key Analysis Performed

##1️⃣ Data Cleaning
* Handled missing values
* Removed duplicates
* Converted data types

## 2️⃣ Exploratory Data Analysis (EDA)
* Distribution of restaurants by location
* Cost analysis
* Rating distribution
* Popular cuisines

##3️⃣ Visualization
* Bar plots
* Pie charts
* Heatmaps
* Distribution plots

## Key Insights
* Certain locations have a higher concentration of restaurants
* Popular cuisines dominate customer choices
* Higher-rated restaurants tend to have more votes
* Pricing varies significantly across locations
